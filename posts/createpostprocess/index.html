2024/05/14

· 本文阅读量次
  1. 使用命令或者窗口新建文章
    hugo new posts/createPostProcess.md

新建的文章默认根目录content目录,因此新建文章相对路径会是conent/posts/createPostProcess.md

  1. 使用编辑器打开创建的文章并编辑 这里使用的编辑器是VSCODE

  2. 重启服务器（可选）

    hugo server

hugo服务器具有热加载功能，一些静态文件才需要重启路由器

  1. 将Markdown生成Html文件 需要将Markdown生成为html文件才能展示在网页上
  hugo

hugo命令就是做这一步。生成的html文件存放与public文件夹

  1. 进入public文件夹，提交至远程仓库即可
    git add .
    git commit -m "xxx"
    git push origin main
